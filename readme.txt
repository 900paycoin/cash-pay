It's a WooCommerce extension allow clients make payment with JPesa.

== Description ==
Abdi Joseph integrated JPesa Payment Gateway with WooCommerce. It is tightly integrated with WooCommerce Core API. When order is placed, user is redirected to JPesa hosted payment page. 

= Features =
* Itemized Checkout - will display each item with SKU/ID
* Enable/Disable Test Mode

= Signup with JPesa =
[Create JPesa Account](https://my.jpesa.com/?dad=user&jc=activate&a=form)

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, you can set options from `WooCommerce -> Settings -> Checkout` menu
